# AgriSmart - Smart Farming Support System

A modern, professional agriculture support website designed to help farmers with planning, weather information, profit prediction, and government scheme access.

## Features

### 1. Login & Registration
- Clean authentication interface
- Full-screen farming background
- Secure login and registration forms

### 2. Home Page
- Welcome banner with inspiring slogan
- Quick access feature cards
- Responsive navigation menu

### 3. Weather Forecast
- Real-time weather updates
- Temperature, humidity, wind speed
- 5-day forecast
- Location-based search

### 4. Market Prices
- Live crop prices
- Multiple location comparison
- Price trend indicators
- Dropdown crop selection

### 5. Seasonal Crop Calendar
- Interactive calendar view
- Color-coded activities (Sowing, Fertilizer, Harvest)
- Month-by-month planning guide

### 6. Farm Profit Predictor
- Calculate expected profits
- Input: Land area, crop type, yield, price
- Output: Production, revenue, profit estimates

### 7. Advanced Farm Calculator
- Seed quantity calculator
- Fertilizer requirement calculator
- Water usage estimator
- Instant results

### 8. Farm Inputs Section
- Seeds, Fertilizers, Pesticides
- Irrigation Equipment
- Agricultural Machinery
- Interactive cards with hover effects

### 9. Government Schemes
- PM-KISAN details
- PMFBY information
- Soil Health Card scheme
- Eligibility and benefits
- Video tutorial section

### 10. Multilanguage Support
- English
- Hindi (हिंदी)
- Tamil (தமிழ்)
- Dynamic language switching

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Data**: JSON files
- **Database**: SQL (schema provided)
- **Icons**: Font Awesome 6.0
- **Design**: Responsive, Mobile-first

## Getting Started

1. Open `index.html` in your browser
2. Login or register (demo mode - redirects to home)
3. Explore all features from the home page
4. Select language from top-right dropdown
